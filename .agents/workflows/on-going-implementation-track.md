## Steps

1. **Creare un file di tracking** nella directory degli artifacts (`C:\Users\Lorenzo\.gemini\antigravity\brain\<conversation-id>`) con nome descrittivo (es. `emom_implementation_progress.md`).
2. **Documentare nello step iniziale** i requisiti e il piano di implementazione.
3. **Aggiornare il file** ad ogni step significativo dell'implementazione con:
   - Cosa è stato fatto
   - File modificati
   - Eventuali problemi riscontrati
   - Prossimi step
4. **Segnare lo stato finale** quando l'implementazione è completa.
